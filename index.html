Ark Grid Core Astrogems Optimizer

Add astrogems with willpower cost and order points, then optimize their distribution into 3 cores with max 4 astrogems each, using tiered thresholds based on core rarity.

How to use this tool:

  1. Add astrogems with their order and willpower cost
  2. Set capacity tier for each core (Epic, Legendary, Relic, or Ancient)
  3. Set priority for each core
  4. Click "Calculate Optimal Cores" to distribute objects
  5. Review the results in the "Optimized Cores" section
